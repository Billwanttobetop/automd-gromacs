## 🔬 技术亮点

### 1. 四合一设计

单个脚本支持四种非平衡方法，通过 `NEMD_TYPE` 参数切换：

```bash
# 余弦加速法（最简单）
NEMD_TYPE=cosine COS_ACCEL=0.1 bash non-equilibrium.sh

# 盒子变形法（最常用）
NEMD_TYPE=deform DEFORM_RATE=0.001 DEFORM_AXIS=xy bash non-equilibrium.sh

# 恒定加速法（最灵活）
NEMD_TYPE=acceleration ACCEL_GROUPS="Protein" ACCEL_X=0.1 bash non-equilibrium.sh

# 壁面驱动法（最真实）
NEMD_TYPE=walls WALL_GROUPS="Wall" WALL_SPEED=0.01 bash non-equilibrium.sh
```

## 🎓 学习要点

### 关键技术

1. **非平衡MD的四种方法**
   - 余弦加速：简单，自动计算粘度
   - 盒子变形：直接模拟剪切
   - 恒定加速：灵活，可控制特定组
   - 壁面驱动：真实边界条件

2. **粘度测量原理**
   - 牛顿流体：η = σ_xy / γ̇
   - 余弦加速：η 自动从速度剖面计算
   - 剪切变稀：η(γ̇) = η₀ / (1 + (λγ̇)ⁿ)

3. **参数选择**
   - 剪切率：γ̇ << 1/τ_relax
   - 简单液体：γ̇ < 0.1 ps⁻¹
   - 聚合物：γ̇ < 0.001 ps⁻¹

4. **GROMACS MDP 参数**
   - `cos-acceleration`: 余弦加速振幅
   - `deform`: 盒子变形速率（6个参数）
   - `acceleration-grps` + `accelerate`: 恒定加速
   - `free-energy` + `delta-lambda`: 壁面驱动

### 物理背景

1. **非平衡态**：系统偏离热力学平衡，存在宏观流动
2. **输运性质**：粘度、热导率、电导率
3. **线性响应**：小驱动力下，响应与驱动成正比
4. **非线性效应**：大驱动力下，出现剪切变稀等现象

```bash

INPUT_TOP=topol.top \

SIM_TIME=1000 \
bash non-equilibrium.sh

# 2. 剪切流动模拟（盒子变形法）
INPUT_GRO=polymer.gro \
INPUT_TOP=topol.top \
NEMD_TYPE=deform \
DEFORM_RATE=0.001 \
DEFORM_AXIS=xy \
SIM_TIME=5000 \
bash non-equilibrium.sh

# 3. 驱动蛋白质流动（恒定加速法）

INPUT_TOP=topol.top \
NEMD_TYPE=acceleration \
ACCEL_GROUPS="Protein" \
ACCEL_X=0.05 \
SIM_TIME=2000 \
```
