## 🎓 应用场景

### 主要应用

1. **配体解离** (distance + constant-velocity)
   - 计算配体-蛋白质解离力
   - 生成伞状采样初始构象
   - 估算结合强度

2. **蛋白质展开** (distance/direction + constant-velocity)
   - 研究蛋白质机械稳定性
   - 识别展开路径
   - 计算展开力

3. **PMF 预备** (constant-velocity → umbrella sampling)
   - 快速生成反应坐标
   - 自动生成窗口配置
   - 无缝对接伞状采样

4. **力响应研究** (constant-force)
   - 研究恒定力下的动力学
   - 计算力-时间响应
   - 识别力诱导的构象变化

## ⚠️ 已知限制

1. **Token 略超目标** (~2700 vs ≤2500)
   - 影响: 轻微增加 API 成本 (+8%)
   - 优化空间: 减少 MDP 注释，简化输出过滤

2. **多组拉力不支持** (pull-ncoords=1)
   - 影响: 无法同时施加多个拉力
   - 解决方案: 手动编辑 MDP 文件
