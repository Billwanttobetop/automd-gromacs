# AutoMD-GROMACS v4.0.0

## Release Summary

AutoMD-GROMACS v4.0.0 expands the project from a workflow-and-sampling toolkit into a broader AI-friendly molecular simulation workbench.

### What is new

- Adds enhanced sampling and special-system workflows including replica exchange, metadynamics, steered MD, enhanced sampling, accelerated MD, coarse-grained simulation, electric-field simulation, non-equilibrium simulation, and QM/MM
- Adds advanced analysis modules for trajectory, binding, property, membrane, scattering, free-energy, and protein-focused studies
- Adds publication-oriented visualization workflow for figure generation and reporting
- Updates package metadata and repository documentation for the expanded AutoMD-GROMACS scope

### Positioning

- Core workflow automation
- Advanced sampling and special scenarios
- Analysis and visualization in the same toolkit
- AI-oriented troubleshooting and token-efficient structure

### Recommended headline

AutoMD-GROMACS v4.0.0: from automated GROMACS workflows to a more complete simulation, analysis, and publication pipeline.
