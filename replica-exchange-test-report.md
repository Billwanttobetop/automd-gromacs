## 发现的问题与修复

### Bug #1: log() 函数污染返回值 ✅ 已修复

**问题描述**:
```bash
# 错误的输出
TEMPS=($(calculate_temperatures 300 350 4))
# 结果: TEMPS=([2026-04-08 09:36:54] 计算温度分布 (指数分布) 300.00 312.00 ...)
```

**根本原因**:
- `calculate_temperatures()` 函数内部调用 `log()` 输出调试信息
- `log()` 输出到 stdout，混入了函数的返回值
- 导致温度数组包含日志时间戳

**修复方案**:
```bash
# 修改前
log() {
    echo "[$(date '+%Y-%m-%d %H:%M:%S')] $*"
}

# 修改后
log() {
    echo "[$(date '+%Y-%m-%d %H:%M:%S')] $*" >&2
}
```

**验证**:
- ✅ 温度数组不再包含日志信息
- ✅ MDP 文件中 `ref_t` 值正确
- ✅ 日志仍然正常输出到 stderr
